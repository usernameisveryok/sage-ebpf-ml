The Problem

  • ▸ eBPF/XDP enables µs-latency packet processing in the Linux kernel
  • ▸ But: no float, 512B stack, 1M instruction limit
  • ▸ Multiply instructions are expensive and dominate inference cost

Can we eliminate ALL multiplications from neural network inference?
